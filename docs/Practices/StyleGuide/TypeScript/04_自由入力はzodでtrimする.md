# 自由入力のテキストは zod で前後の空白文字を trim() する

## 概要

- フォームなどでユーザが自由入力する `string` 型のテキストは、zod の `trim()` を使って前後の半角・全角スペースを取り除く。

## 👎 不適切な例

```ts
memo: z.string().min(1).max(MAX_MEMO_LENGTH)
```

## 👍 適切な例

```ts
memo: z.string().trim().min(1).max(MAX_MEMO_LENGTH)
```

## なぜこのルールが必要か

半角・全角スペースのみの文字列をユーザが送信することを防ぐため。`min(1)` の前に `trim()` を挟むことで、前後の空白が取り除かれた結果が空文字になる入力を弾ける。
