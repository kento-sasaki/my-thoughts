# 道具が我々を形作る


私はある時期、スマホに時間を奪われている自覚があった。ちょっとした隙間に手に取って SNS や YouTube を眺め、寝る前と起きた直後はベッドでだらだら画面を見る。本当はやりたいこと（英語学習、読書、運動）があるのに、小さな浪費の積み重ねで時間が溶けていく。

そこで、いくつか手を打った。この個人的な試行錯誤が、結局はデザインと技術倫理の話に繋がっている。

## 時間を取り戻すために打った手

- **使用状況のモニタリング** — どのアプリにどれだけ時間を使い、何回ロックを解除し、何件通知が来ているかを把握する。まず「何に奪われているか」を知る。
- **アプリタイマー** — アプリごとに1日の利用上限を設定する。すると「予算」の心理が生まれ、「この30分は無駄遣いしない」と考えるようになる。
- **サイレントモード＆モノクロ化** — 本当に必要な通知以外を遮断し、画面を白黒にする。色を奪うだけで「見たい」衝動がかなり減る。サイレントモードのオンが「集中モード」のメンタルスイッチにもなった。
- **AppBlock で時間帯制限** — 就寝前〜起床後（23:30〜8:00）は特定アプリを開けなくする。これは効果絶大だった。
- **マインドセット** — 「我慢」ではなく「自分の時間を取り戻す儀式」と捉え直す。制限は外そうと思えば外せるが、この捉え方のおかげで、制限を自分のためのものだと思える。

## 気づいたこと：道具は中立ではない

一連の対策のなかで、ひとつ象徴的な発見があった。**画面を白黒にするだけで、スマホが本当につまらなく見える。** なぜか。SNS は色鮮やかな写真や動画で、こちらの注意を引くように作られているからだ。色を奪うとその引力が弱まる。

つまり、私がスマホに時間を奪われていたのは、意志が弱いからというより、**そう作られているから**だった。無限スクロール、通知、鮮やかな配色、レコメンド。これらはすべて、ユーザーの注意を最大化するよう設計された UI だ。道具は中立ではない。使う人の行動と注意を、確かに形作っている。

「我々は道具を形作り、道具は我々を形作る」。スマホ依存と戦う体験は、この後半、つまり道具が我々を形作る側面を、身をもって教えてくれた。

## 作り手に回ったとき、それは責任になる

ここからが本題だ。私はスマホの「ユーザー」としてこの非中立性に振り回された。だが、エンジニアである私は同時に、道具を「作る側」でもある。

[UI は認知のフレームを設計している](./01_UIは認知フレーム.md)のだとすれば、その力は両刃だ。ユーザーの期待を尊重し、やりたいことを助けるフレームも作れるし、注意を奪い、滞在時間を引き延ばし、本来の目的から逸らすフレームも作れる。後者を意図的に設計したものを、私たちはダークパターンと呼ぶ。

だから、作り手であることは責任を伴う。「エンゲージメントを上げる」「滞在時間を伸ばす」という指標は、しばしばユーザーの時間を奪う設計と地続きだ。指標を追うこと自体が悪いのではなく、**その指標がユーザーを良い方向へ形作っているのか、ただ注意を搾取しているのかを問わないこと**が問題なのだと思う。

私がスマホから時間を取り戻すために色を消したように、作り手は逆に、色や通知やスクロールでユーザーの行動を引き出せる。その力を持っていると自覚すること。そして、自分が作る道具が人をどう形作るかを引き受けること。これが、エンジニアがデザインを、そして技術倫理を考えるべき理由だと私は思っている。

道具を形作る私たちは、その道具が形作る人々に対して、無関係ではいられない。
